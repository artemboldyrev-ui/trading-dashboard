Trading Dashboard

Pripoj Google Drive, vyber CSV súbory z priečinka Trading sheets a grafy sa vygenerujú automaticky.

Google Client ID
ID priečinka na Drive (voliteľné)

Appka beží len v tvojom prehliadači. Dáta sa nikam neposielajú.
Potrebuješ Client ID z Google Cloud Console — návod nižšie.

── Návod na nastavenie (1× stačí) ──
01
Otvor Google Cloud Console
Choď na console.cloud.google.com a prihlás sa svojím Google účtom.
02
Vytvor nový projekt
Hore vľavo klikni na selector projektov → New Project → pomenuj ho napr. trading-dashboard → Create.
03
Aktivuj Google Drive API
V ľavom menu: APIs & Services → Enable APIs → vyhľadaj Google Drive API → Enable.
04
Vytvor OAuth 2.0 Client ID
APIs & Services → Credentials → Create Credentials → OAuth Client ID → Application type: Web application.

Do poľa Authorized JavaScript origins pridaj URL tejto stránky (napr. file:// ak ju otváraš lokálne, alebo tvoju doménu).
05
Skopíruj Client ID
Po vytvorení uvidíš Client ID — skopíruj ho a vlož do poľa vyššie.
06
Nastav OAuth consent screen
OAuth consent screen → User Type: External → vyplň názov appky → v sekcii Scopes pridaj drive.readonly → v Test users pridaj svoj email.
07
ID priečinka Trading sheets
